# Overview
The Etch-a-Sketch project is a web application that simulates the classic drawing toy, enabling users to create interactive drawings on a digital canvas. This project is part of The Odin Project foundation curriculum.

# Features
* **Dynamic grid generation**: Create custom grids by inputting the number of rows/columns.
* **Colorful drawing effects**: Hover over grid squares to change their colour.
* **Clear functionality**: Reset the canvas and create a new grid with a single button press.
## Extra Features:
* Square darkening with repeated interactions.

# Installation Guide
1. Clone the repository:
```sh
git clone https://github.com/0xmueed/etch-a-sketch.git
```
2. Navigate to project directory:
```sh
cd etch-a-sketch
```
3. Open in browser: Open `index.html` in your preferred browser.
